# Mable MarketPlace Project

Mable MarketPlace is an online retailer that operates in South Africa and has recently started doing bbusiness internationally. This project explores seasonal trends of Mable's sales data and historic customer information, and uses regression techniques to build a model to predict the sales of a client. The model is then used to intepret which factors are relates to Sales. 

This project uses Python tools/libraries such as pandas, numpy, scikitlearn, seaborn, and matplotlib, and runs on Jupyter Notebook.
